Password Generator (Python)
## Description

A Python-based password generator that creates secure and customizable passwords. Users can generate passwords with optional digits and special characters, along with a strong password mode that ensures high security by including a mix of uppercase, lowercase, numeric, and special characters.

# Features
Generate custom passwords based on user preferences
Option to include digits and special characters
Strong password generator with guaranteed complexity
Randomized and secure password creation
Simple command-line interface
# Technologies Used
Python
Random Module
String Module
# How It Works

The program uses Python’s random and string modules to generate passwords.

Custom mode allows flexible character selection
Strong mode ensures at least one character from each category (A-Z, a-z, 0-9, symbols)
Password is shuffled for better randomness and security
# How to Run
Install Python (if not already installed)
Copy the code into a .py file (e.g., password_generator.py)

Run the file:

python password_generator.py
Choose an option and enter desired length
